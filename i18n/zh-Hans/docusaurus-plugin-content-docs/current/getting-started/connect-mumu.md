# Connect MuMu to Microbe Studio

Open Microbe Studio and confirm that the MuMu instance appears in the instance list.

![Instance visible in Microbe Studio](/img/getting-started/setup-mumu/17-microbe-instance-visible.webp)

If no instance appears, open **Settings** in Microbe Studio and check the MuMu CLI path.

![Microbe Studio settings](/img/getting-started/setup-mumu/18-microbe-settings-entry.webp)

The default path is usually:

```text
C:\Program Files\Netease\MuMuPlayer\nx_main\mumu-cli.exe
```

If MuMu is installed on another drive, the path is similar. For example:

```text
D:\Program Files\Netease\MuMuPlayer\nx_main\mumu-cli.exe
```

![MuMu CLI path setting](/img/getting-started/setup-mumu/19-mumu-cli-path-setting.webp)
