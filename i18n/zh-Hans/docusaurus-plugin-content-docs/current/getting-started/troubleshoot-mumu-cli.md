# Troubleshoot `mumu-cli.exe`

Use this page when Microbe Studio cannot detect MuMu automatically.

## Find `mumu-cli.exe`

On Windows, search for **MuMuPlayer**, then choose **Open file location**.

![Search MuMuPlayer](/img/getting-started/setup-mumu/20-windows-search-mumu.webp)

Right-click the MuMuPlayer shortcut and choose **Open file location** again if needed.

![Open file location](/img/getting-started/setup-mumu/21-open-file-location.webp)

Find `mumu-cli.exe` in the MuMu installation folder.

![Find mumu-cli.exe](/img/getting-started/setup-mumu/22-find-mumu-cli.webp)

If it is hard to find, search for `mumu-cli.exe` inside the folder.

![Search mumu-cli.exe](/img/getting-started/setup-mumu/23-search-mumu-cli.webp)

Right-click `mumu-cli.exe`, open **Properties**, and go to the **Security** tab.

![Open properties](/img/getting-started/setup-mumu/24-mumu-cli-properties.webp)

![Security tab](/img/getting-started/setup-mumu/25-properties-security-tab.webp)

Copy the full path from **Object name**.

![Copy object name](/img/getting-started/setup-mumu/26-copy-object-name.webp)

Paste that path into the MuMu CLI path field in Microbe Studio.

![Paste CLI path](/img/getting-started/setup-mumu/27-paste-cli-path.webp)

Close and reopen Microbe Studio. When the instance appears, the setup is complete.

![Instance detected](/img/getting-started/setup-mumu/28-instance-detected.webp)
