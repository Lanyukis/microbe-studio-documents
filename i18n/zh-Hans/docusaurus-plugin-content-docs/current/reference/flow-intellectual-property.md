# Flow 知识产权

本页面说明 Microbe Studio 平台与自动化流程之间的知识产权归属。本文中的自动化流程统称为 **Flow**。

:::note
本文是产品文档中的补充说明。Microbe Studio 的使用仍受主网站上的官方 [Terms of Service](https://microbe-studio.com/terms)、[Privacy Policy](https://microbe-studio.com/privacy) 和 [Refund Policy](https://microbe-studio.com/refund) 约束。
:::

## 0. 与官方政策的关系

本页面中与 Flow 相关的内容应与以下官方政策一并阅读：

- [Terms of Service](https://microbe-studio.com/terms)：约束 Service、Token、License Package、Protected Flow 和知识产权的使用条款。
- [Privacy Policy](https://microbe-studio.com/privacy)：说明 Microbe Studio 如何处理账号、设备、license、交易和技术数据。
- [Refund Policy](https://microbe-studio.com/refund)：说明 Token、License Package 和 Protected Flow 的退款条件。

如果本页面与主网站上的官方政策存在差异，以主网站上的官方政策为准。

## 1. Microbe Studio 的所有权

Microbe Studio 拥有平台相关的全部知识产权，包括但不限于：

- 软件源代码。
- 系统架构和平台结构。
- 应用界面。
- API 和集成机制。
- 流程构建工具，包括 Flow Builder。
- 用于运行、测试、存储和执行 Flow 的核心软件组件。

除非另有书面协议，上述权利归 Microbe Studio 所有，不转让给 Partner、客户或任何第三方。

## 2. Partner 对 Flow 的所有权

在本文中，**Flow** 包括 Partner 在 Microbe Studio 平台上自行配置的场景设计、配置逻辑、模板内容、节点配置以及步骤排列方式。

Partner 创建的 Flow 的知识产权归该 Partner 独家所有。未经 Partner 书面同意，Microbe Studio 不得擅自复制、出售、分发或转让这些 Flow 给任何第三方。

## 3. 授权终端客户使用 Flow

当 Partner 向终端客户销售或提供 Flow 时，Partner 授予终端客户在 Microbe Studio 平台上使用该 Flow 的权利。

该使用权具有以下限制：

- 非独占。
- 不可转让。
- 仅限于指定账号、Workspace 或组织范围内使用。
- 有效期与终端客户的软件订阅期限或终端客户与 Partner 之间的商业协议期限一致。

未经 Partner 书面同意，终端客户不得以商业目的复制、转售、再次分发 Flow，或将 Flow 的访问权转让给其他账号、个人或组织。

## 4. 服务构成

客户使用 Microbe Studio 时，可能会接触到两个独立的知识产权构成部分：

| 构成部分 | 所有者 | 说明 |
| --- | --- | --- |
| 平台 | Microbe Studio | 源代码、界面、API、Flow Builder、执行机制和平台工具。 |
| Flow | Partner | Partner 配置的逻辑、处理步骤、模板内容、节点配置和流程排列。 |

客户获得 Flow 的使用权，并不代表客户取得 Flow 的所有权，也不代表客户取得 Microbe Studio 平台的所有权。

## 5. 与 Flow 相关的禁止行为

客户承诺不直接或间接实施以下行为：

### 5.1. 复制 Flow 结构

客户不得通过截图、录屏、记录逻辑结构，或使用技术工具复制 Flow 配置，以便在其他账号或其他平台上重新创建该 Flow。

### 5.2. 未经授权的商业化

客户不得在授权范围之外，向任何个人或组织转售、出租、赠与、分发或转让 Flow 的激活码、访问权或使用权。

### 5.3. 逆向工程

未经 Partner 和 Microbe Studio 书面同意，客户不得故意干预源代码、API、配置数据或平台机制，以提取、重建或复现 Flow 的数据结构。

## 6. 技术保护机制

Microbe Studio 可以为 Partner 标记为 **Exclusive** 的 Flow 建立技术保护机制，包括但不限于：

- 加密 Flow 数据。
- 隐藏 Flow 的详细结构。
- 限制查看、复制或导出 Flow 数据。
- 按账号、Workspace 或组织控制访问权限。

这些机制旨在减少客户账号或其他 Partner 对 Partner 所有 Flow 的未经授权复制。

## 7. 终止使用权

如果客户违反本页面规定，或终止与 Partner 的服务协议，Microbe Studio 可以根据拥有该 Flow 的 Partner 的请求或平台政策：

- 撤销客户对 Flow 的访问权。
- 冻结或删除客户 Workspace 中的违规 Flow。
- 暂停或终止客户对违规 Flow 相关功能的访问。

如果违规由客户造成，剩余订阅期限对应的软件费用可能不予退还，具体取决于适用的商业条款。

## 8. 赔偿责任

如果客户非法复制、分发或商业化 Partner 的 Flow，并造成商业损失，客户应负责赔偿 Partner 实际发生的损失。

除赔偿责任外，客户还可能根据越南现行知识产权法律以及各方之间的相关协议承担相应处理措施。
