# Microbe Studio Documentation

Microbe Studio is a Windows desktop app for serious automation on Android emulator instances, with iOS support through MuMu when the emulator mode supports it.

This documentation is organized around two jobs:

- **Using the app**: install the required tools, prepare emulator instances, run flows, manage queues, schedule work, and use the Discord bot.
- **Building flows**: create, edit, validate, test, and reuse flows in the visual node editor.

The current writing workflow starts in Vietnamese first, then the stable content is translated into English and Simplified Chinese. English remains the public fallback language for the docs site.
