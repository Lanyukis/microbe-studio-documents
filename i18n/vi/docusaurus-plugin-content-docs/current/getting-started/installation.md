# Cài đặt

Hướng dẫn cài đặt Microbe Studio và setup môi trường.

## Bạn cần chuẩn bị

- Máy tính hệ điều hành Windows.
- Truy cập và tải App tại đây: https://microbe-studio.com/download

## Cài đặt Microbe-Studio

Tải file [**MicrobeStudio_Setup**](https://microbe-studio.com/download) và chạy file cài đặt.

![ảnh more info](/img/intro/more-info.png)
Hãy Click vào **More Info** để tiếp tục

![ảnh](/img/intro/run-anyways.png)
Click **Run anyways** để tiến hành cài đặt

![ảnh](/img/intro/finish-install.png)
Như vậy là hoàn thành cài đặt


## Bước tiếp theo

Trước khi chạy App Microbe-Studio, hãy setup MuMu theo hướng dẫn sau:

[Cách Setup MuMu](./setup-mumu)

## Lưu ý iOS

Microbe Studio có thể hỗ trợ iOS khi workflow chạy qua MuMu. Tạm thời tài liệu Android và iOS dùng chung, chỉ tách riêng khi test thực tế cho thấy UI hoặc flow khác đáng kể.
