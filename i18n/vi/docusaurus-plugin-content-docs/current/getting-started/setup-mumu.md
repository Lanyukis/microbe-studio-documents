# Cách Setup MuMu để sử dụng Microbe Studio

Microbe Studio chạy trực tiếp với giả lập MuMu. Trước khi chạy flow, hãy cài và cấu hình MuMu để app có thể nhận hoạt động ổn định.

## Các bước setup

1. [Cài đặt MuMu](./install-mumu)
2. [Cấu hình MuMu](./configure-mumu)
3. [Kết nối MuMu với Microbe Studio](./connect-mumu)
4. [Xử lý lỗi `mumu-cli.exe`](./troubleshoot-mumu-cli)

## Cấu hình khuyến nghị

- Bật VT để tối ưu hiệu năng giả lập.
- Đặt cấu hình giả lập là **2 cores**, **2 GB RAM**, **1280x720**, **240 DPI**, và **30 FPS** để flow đồng bộ và ổn định hơn.
- Restart MuMu sau khi thay đổi cấu hình.
- Restart Microbe-studioo sau khi thay đổi cấu hình.
