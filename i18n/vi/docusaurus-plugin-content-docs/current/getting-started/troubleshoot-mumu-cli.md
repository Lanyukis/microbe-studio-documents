# Xử lý lỗi `mumu-cli.exe`

Dùng trang này khi Microbe Studio không tự detect được MuMu.

## Cách lấy đường dẫn `mumu-cli.exe`

Trên Windows, tìm **MuMuPlayer**, sau đó chọn **Open file location**.

![Tìm MuMuPlayer](/img/getting-started/setup-mumu/20-windows-search-mumu.webp)

Nếu mở ra shortcut, chuột phải vào MuMuPlayer và chọn **Open file location** thêm một lần nữa.

![Open file location](/img/getting-started/setup-mumu/21-open-file-location.webp)

Tìm file `mumu-cli.exe`.

![Tìm mumu-cli.exe](/img/getting-started/setup-mumu/22-find-mumu-cli.webp)

Nếu không thấy, có thể tìm `mumu-cli.exe` trên thanh tìm kiếm trong thư mục.

![Search mumu-cli.exe](/img/getting-started/setup-mumu/23-search-mumu-cli.webp)

Chuột phải vào file, chọn **Properties**, rồi mở tab **Security**.

![Properties](/img/getting-started/setup-mumu/24-mumu-cli-properties.webp)

![Security tab](/img/getting-started/setup-mumu/25-properties-security-tab.webp)

Copy đầy đủ đường dẫn ở mục **Object name**.

![Object name](/img/getting-started/setup-mumu/26-copy-object-name.webp)

Dán đường dẫn vào ô CLI trong Microbe Studio.

![Dán CLI path](/img/getting-started/setup-mumu/27-paste-cli-path.webp)

Tắt Microbe Studio và mở lại app. Khi thấy instance hiển thị, setup đã hoàn tất.

![Instance đã hiển thị](/img/getting-started/setup-mumu/28-instance-detected.webp)
