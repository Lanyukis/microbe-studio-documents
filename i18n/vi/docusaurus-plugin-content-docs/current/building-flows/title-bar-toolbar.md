# Title Bar Toolbar

Title Bar chứa các công cụ canvas và action cấp flow.

## Nội dung sẽ viết

- Undo, Redo, Zoom Out, Zoom In.
- Fit Screen, Fit Selection, Snap To Grid, MiniMap, Auto Layout.
- Copy selected node và Paste node.
- Flow Input, Max node execution, Save, Test Flow, Validate, Export, Import.
