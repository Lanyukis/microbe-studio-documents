# Node Properties Panel

Node và Region Properties panel dùng để cấu hình node hoặc vùng màn hình đang được chọn.

## Nội dung sẽ viết

- Tabs Properties, Validation, Assets và Runtime.
- Test Node, Delete node, Duplicate node, Reset và Apply.
- Select Directly On Live Preview.
- Capture Template và Save Region.
