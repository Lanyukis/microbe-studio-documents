# Instance Dashboard

Instance Dashboard dùng để quản lý các giả lập và theo dõi instance nào đang chạy flow nào.

## Nội dung sẽ viết

- Thêm và nhận diện instance MuMu.
- Start, Pause và Shutdown instance.
- Đọc trạng thái Online / Offline.
- Hiểu Instance ID, Run Evidence, Flow Queue, Current Flow và Next in Queue.

## Raw notes cần bổ sung

- App phản hồi thế nào khi instance offline.
- Ảnh chụp từng trạng thái instance.
- Khác biệt nếu có giữa Android và iOS mode trong MuMu.
