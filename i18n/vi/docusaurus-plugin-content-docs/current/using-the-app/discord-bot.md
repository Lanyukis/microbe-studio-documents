# Discord Bot

Discord Bot cho phép chạy task hoặc flow trực tiếp bằng command từ Discord mà không cần mở Microbe Studio.

## Nội dung sẽ viết

- Kết nối bot vào Discord server.
- Cấu hình quyền và role.
- Danh sách command, cú pháp và ví dụ.
- Hành vi queue, chạy song song và phản hồi lỗi.
