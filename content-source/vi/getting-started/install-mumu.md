# Cài đặt MuMu

Tải MuMu tại:

https://www.mumuplayer.com/

![Trang tải MuMu](/img/getting-started/setup-mumu/01-mumu-download-page.webp)

Mặc định hãy chọn bản Windows. Chỉ dùng bản Windows ARM nếu máy của bạn chạy chip ARM.

Sau khi tải file `.exe`, mở file và nhấn **Install Now**.

![Cài đặt MuMu](/img/getting-started/setup-mumu/02-install-now.webp)

Sau khi cài xong, mở MuMu và kiểm tra giao diện chính.

![Giao diện MuMu](/img/getting-started/setup-mumu/03-mumu-home.webp)
