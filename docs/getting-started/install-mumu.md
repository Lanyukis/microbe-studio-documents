# Install MuMu

Download MuMu Player from the official site:

https://www.mumuplayer.com/

![MuMu download page](/img/getting-started/setup-mumu/01-mumu-download-page.webp)

Choose the installer for your operating system. For most users, the default Windows version is correct. Use the Windows ARM version only if your Windows device uses an ARM chip.

Open the downloaded `.exe` file and select **Install Now**.

![Install MuMu](/img/getting-started/setup-mumu/02-install-now.webp)

After installation, open MuMu and confirm that the main interface loads.

![MuMu home screen](/img/getting-started/setup-mumu/03-mumu-home.webp)
