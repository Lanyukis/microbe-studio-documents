# Glossary

Use this page to keep terms consistent across Vietnamese, English, and Chinese docs.

| Term | Meaning |
| --- | --- |
| Instance | One emulator instance controlled by Microbe Studio. |
| Flow | A reusable automation workflow. |
| Node | One step or decision point inside a flow. |
| Region | A selected screen area used by detection or OCR. |
| Template | A captured image or asset used for matching. |
| Sub Flow | A flow called from another flow. |
| Shared Sub Flow | A reusable sub flow shared by multiple flows. |
| Flow Input | Parameters passed into a flow before execution. |
| Max node execution | A safety limit that stops repeated node execution. |
| MuMu | The emulator used by Microbe Studio. |
